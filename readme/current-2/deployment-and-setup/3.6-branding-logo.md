# 3.6. ຍີ່ຫໍ້ (ໂລໂກ້)

ໂລໂກ້ທີ່ສະແດງຢູ່ມຸມຊ້າຍເທິງຂອງ GeoPrism Registry ແມ່ນສາມາດຕັ້ງຄ່າໄດ້ໂດຍຜູ້ບໍລິຫານລະບົບ (System Administrator).

1.  ໄປທີ່ໜ້າການຕັ້ງຄ່າ (Setting) ຈາກເມນູ (ຮູບ 3 ຂີດແຈມູມຂວາ).
