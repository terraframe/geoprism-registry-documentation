# 6.2.1. ຂໍ້ມູນຮັບຮອງ local GeoPrism Registry

1.  ໃນໜ້າເຂົ້າສູ່ລະບົບ GeoPrism Registry, ພິມຊື່ຜູ້ໃຊ້ ແລະ ລະຫັດຜ່ານທີ່ກໍາໜົດໃຫ້ທ່ານໃນຊ່ອງທີ່ເໝາະສົມ.
